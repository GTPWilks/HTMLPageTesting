to scrollDown_atBottom scrollArea
 
Will read text in a given area, scroll, then read text again to see if scrolling changed what is displayed. If no change then assumption is made at bottom
 Params scrollArea -- Predefined screen area defined in ScreenArea scipt. This will be the area to check if scrolling changed anything (via OCR)
 Returns boolean true/false. True is bottom of scroll area is reached
 Snippet Navigate.scrollDown_atBottom [scrollArea]
 Tag @@Navigate.scrollDown_atBottom, @@Scroll, @@Bottom
to validateScreenSettled areaToRead, timeToWait
 
Will read text in given area of screen and store as a variable. Then will read the screen again and validate the two values are equal
If equal then no new text is in the area and assuming screen has settled, if new values then will repeat until equal or failure/timeout happens
 Params areaToRead -- Predefined screen area defined in ScreenArea scipt
timeToWait -- approx time of long to check screen has settled before just exiting -- default is 15
 Returns boolean - true/false. Screen has settled
 Snippet Navigate.validateScreenSettled [areaToRead], [timeToWait]
 Tag @@Navigate.validateScreenSettled, @@Validate, @@Settled
