Script_Name: Vault
Purpose: Handler scripts for interacting with Windows Credential Manager

Can call this script directly to create a token
If in EPM, Create a new Test called "azureToken" and link the Framework_V2 Vault script to it. In the parameter for the script put your token value (in quotes)
Locally you can run as
Vault.credentialManager "Set", "vvnhuvbxd3wfrdmq6ednhptrvckilq364g2mdeyaqnov56a75naq" -- replace with your token (value here is not a valid token)
